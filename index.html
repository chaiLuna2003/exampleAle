Objetivo

Crear un sitio web responsivo que se adapte a diferentes dispositivos y tamaños de pantalla, utilizando HTML y CSS y un poco de JavaScript. El sitio debe incluir un menú de navegación, una sección de contenido principal y un pie de página. Además, se deben aplicar estilos CSS para mejorar la apariencia visual del sitio. para esto primero deberemos de crear y trabajar con una secion de navegacion por lo que se cuenta por al menos una semana para el diseño
